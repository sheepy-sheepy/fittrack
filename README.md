# FitTrack - Fitness Tracking Application

## 🏋️ Описание
FitTrack - это мобильное приложение для отслеживания питания, параметров тела и физической активности.

## ✨ Основные функции
- 🔐 Регистрация и авторизация с верификацией email
- 📊 Онбординг с расчетом процента жира
- 🍽️ Дневник питания с учетом КБЖУ и воды
- 📏 Ежедневный ввод параметров тела
- 📝 CRUD операции с рецептами
- 🛒 CRUD операции с продуктами
- 📸 Добавление 4 фото в день
- 📈 Аналитика с графиками изменений
- ☁️ Синхронизация с облаком (Supabase)
- 💾 Локальное хранение данных (Drift)

## 🛠️ Технологии
- Flutter 3.0+
- Supabase (Backend & Auth)
- Drift (Local Database)
- Provider (State Management)
- Fl Chart (Graphs)
- Image Picker (Photos)
